##Синтаксис
  Z - нулевая функция; Z(x) = 0.
  S - прибавить 1; S(x) = x + 1.
  P<n,k> - проектор(k-ый аргумент из полученных n аргументов)
  С(f, g1, g2, ...) - композиция функций(f - главная)
  R(f, g) - рекурсия
